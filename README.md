# Nakshatra

Nakshatra is a hardware-integrated, secure device authentication framework for IoT devices.

## Structure
- `client/`: Code for IoT client devices (ESP32, Raspberry Pi).
- `frontend/`: React.js web interface.
- `backend/`: Centralized authentication server (Python).
- `broker/`: MQTT Mosquitto broker setup.
